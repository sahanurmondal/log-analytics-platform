### Pattern 4: BST Construction from Sorted Input
```java
TreeNode buildBST(int[] nums, int left, int right) {
    if (left > right) return null;
    int mid = left + (right - left) / 2;
    TreeNode root = new TreeNode(nums[mid]);
    root.left = buildBST(nums, left, mid - 1);
    root.right = buildBST(nums, mid + 1, right);
    return root;
}
```

## Interview Tips

### Common Mistakes to Avoid
1. **Boundary Conditions**: Handle null nodes properly
2. **BST Property**: Remember left < root < right (not <=)
3. **Integer Overflow**: Use Long for large value comparisons
4. **Edge Cases**: Single node, empty tree, duplicate handling

### Optimization Techniques
1. **Early Termination**: Stop traversal when target found
2. **Pruning**: Skip subtrees outside query range
3. **Space Optimization**: Prefer iterative over recursive when possible
4. **Caching**: Store frequently accessed values

### Communication Points
1. **Clarify Requirements**: Ask about duplicates, null handling
2. **Discuss Trade-offs**: Time vs space complexity
3. **Mention Alternatives**: Different approaches and their benefits
4. **Edge Case Handling**: Demonstrate thorough testing mindset
