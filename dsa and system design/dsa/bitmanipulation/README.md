## Key Bit Manipulation Techniques

### Fundamental Operations
1. **XOR Properties**: `a ^ a = 0`, `a ^ 0 = a`, commutative and associative
2. **AND Operations**: `n & (n-1)` removes rightmost 1 bit
3. **OR Operations**: Set specific bits
4. **Bit Shifting**: `<<` for multiplication by 2, `>>` for division by 2
5. **Bit Masking**: Extract or modify specific bits

### Advanced Patterns
1. **Brian Kernighan's Algorithm**: Count set bits efficiently
2. **Two's Complement**: Handle negative numbers
3. **Bit Manipulation DP**: Use bits as state representation
4. **Trie + Bits**: Maximum XOR problems
5. **Gray Code**: Generate sequences with single bit differences

### State Compression Techniques
1. **Bitmask DP**: Represent subsets as integers
2. **Traveling Salesman**: Use bits to track visited cities
3. **Subset Enumeration**: Iterate through all subsets
4. **Game States**: Represent board positions compactly

## Core Bit Manipulation Patterns

### Pattern 1: XOR for Uniqueness
```java
// Find single number among duplicates
int result = 0;
for (int num : nums) {
    result ^= num; // Duplicates cancel out
}
return result;
```

### Pattern 2: Brian Kernighan's Algorithm
```java
// Count set bits efficiently
int count = 0;
while (n != 0) {
    n &= (n - 1); // Remove rightmost 1 bit
    count++;
}
return count;
```

### Pattern 3: Bit Masking
```java
// Check if i-th bit is set
boolean isSet = (n & (1 << i)) != 0;

// Set i-th bit
n |= (1 << i);

// Clear i-th bit
n &= ~(1 << i);

// Toggle i-th bit
n ^= (1 << i);
```

### Pattern 4: Power of Two Check
```java
// Check if number is power of 2
boolean isPowerOfTwo = n > 0 && (n & (n - 1)) == 0;
```

### Pattern 5: Subset Generation
```java
// Generate all subsets using bit manipulation
for (int mask = 0; mask < (1 << n); mask++) {
    List<Integer> subset = new ArrayList<>();
    for (int i = 0; i < n; i++) {
        if ((mask & (1 << i)) != 0) {
            subset.add(nums[i]);
        }
    }
    result.add(subset);
}
```

## Interview Tips & Common Pitfalls

### Essential Knowledge
1. **Binary Representation**: Understand two's complement
2. **Bit Operations**: Master AND, OR, XOR, NOT, shifts
3. **Common Tricks**: `n & (n-1)`, `n & (-n)`, bit counting
4. **Overflow Handling**: Be careful with 32-bit integers

### Optimization Techniques
1. **Early Termination**: Stop when no more bits to process
2. **Bit Length Calculation**: Use `Integer.numberOfLeadingZeros()`
3. **Cache Results**: For repeated calculations
4. **Parallel Processing**: Count bits in multiple positions simultaneously

### Common Mistakes
1. **Sign Extension**: Be careful with negative numbers and right shifts
2. **Integer Overflow**: Use long for intermediate calculations
3. **Bit Position**: Remember 0-indexed bit positions
4. **Edge Cases**: Handle 0, negative numbers, and boundary values
