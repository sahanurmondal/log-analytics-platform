## Key Patterns & Templates

### 1. Basic Backtracking Template
```java
public void backtrack(List<T> result, List<T> current, /* parameters */) {
    // Base case
    if (isValidSolution(current)) {
        result.add(new ArrayList<>(current));
        return;
    }
    
    // Try all possible choices
    for (T choice : getChoices(/* parameters */)) {
        // Choose
        current.add(choice);
        
        // Explore (with updated parameters)
        if (isValid(current, choice)) {
            backtrack(result, current, /* updated parameters */);
        }
        
        // Unchoose
        current.remove(current.size() - 1);
    }
}
```

### 2. Permutations Template
```java
public void permuteBacktrack(List<List<Integer>> result, List<Integer> current, 
                           int[] nums, boolean[] used) {
    if (current.size() == nums.length) {
        result.add(new ArrayList<>(current));
        return;
    }
    
    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        
        current.add(nums[i]);
        used[i] = true;
        permuteBacktrack(result, current, nums, used);
        used[i] = false;
        current.remove(current.size() - 1);
    }
}
```

### 3. Combinations Template
```java
public void combineBacktrack(List<List<Integer>> result, List<Integer> current,
                           int start, int n, int k) {
    if (current.size() == k) {
        result.add(new ArrayList<>(current));
        return;
    }
    
    for (int i = start; i <= n; i++) {
        current.add(i);
        combineBacktrack(result, current, i + 1, n, k);
        current.remove(current.size() - 1);
    }
}
```

### 4. Subsets Template
```java
public void subsetsBacktrack(List<List<Integer>> result, List<Integer> current,
                           int[] nums, int start) {
    result.add(new ArrayList<>(current));
    
    for (int i = start; i < nums.length; i++) {
        current.add(nums[i]);
        subsetsBacktrack(result, current, nums, i + 1);
        current.remove(current.size() - 1);
    }
}
```
